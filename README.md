### Deskripsi Proyek
Proyek ini merupakan implementasi data warehouse menggunakan pendekatan Star Schema berdasarkan dataset Cuaca Harian Kebumen (1978-2024).
Tujuan utama proyek ini adalah untuk mendukung proses analisis data dan pengambilan keputusan berbasis data.

### Tujuan Proyek
Proyek ini dibuat untuk memenuhi tugas Ujian Tengah Semester (UTS) mata kuliah Business Intelligence.

Dataset yang digunakan diambil dari Kaggle:
